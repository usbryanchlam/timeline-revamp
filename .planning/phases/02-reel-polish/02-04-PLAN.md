  <name>Task 5: Write comprehensive stateMachine.test.ts</name>
  <files>src/gestures/stateMachine.test.ts</files>
  <read_first>src/gestures/stateMachine.ts</read_first>
  <action>Create `src/gestures/stateMachine.test.ts`. Cover every transition in `transition()` with explicit named tests. Group by state via `describe`. Tests required (one `it` block per assertion):

  **Lifecycle events (top-priority, work in any state):**
  - VIS_HIDDEN: any state → SUSPENDED (test from each of IDLE, SCRUBBING, MAP_INTERACT, PAUSED)
  - VIS_VISIBLE: SUSPENDED → IDLE; non-SUSPENDED is no-op
  - ORIENTATION_CHANGE: any non-SUSPENDED state → SUSPENDED with scrubT cleared
  - ORIENTATION_SETTLE: SUSPENDED → IDLE; non-SUSPENDED is no-op

  **POINTER_DOWN:**
  - 1-finger from IDLE: stays IDLE name, sets pointerCount=1, gestureStartedAt set
  - 2-finger from IDLE: → MAP_INTERACT, pointerCount=2
  - 2-finger from SCRUBBING: → MAP_INTERACT (cancels scrub, scrubT=null)
  - 2-finger from PAUSED: → MAP_INTERACT

  **LONG_PRESS_FIRED:**
  - 1-finger held in IDLE → SCRUBBING with scrubT initialized to chapterIndex/(total-1)
  - 1-finger held in PAUSED → SCRUBBING
  - From CHAPTER_SWIPE: no-op (not IDLE/PAUSED)
  - With pointerCount !== 1: no-op

  **POINTER_MOVE:**
  - In SCRUBBING with single finger: scrubT updates from event.dx, clamped to 0..1
  - In MAP_INTERACT with 2 fingers: stays MAP_INTERACT, accumulates dx/dy
  - In IDLE with 1 finger: accumulates dx/dy, doesn't change state

  **POINTER_UP — flick path:**
  - SingleFinger in IDLE, gestureDy < -30 AND duration < 300ms: → CHAPTER_SWIPE with chapterIndex+1, clamped at total-1
  - SingleFinger in IDLE, gestureDy > 30 AND duration < 300ms: → CHAPTER_SWIPE with chapterIndex-1, clamped at 0
  - SingleFinger in IDLE, gestureDy >= 30 BUT duration >= 300ms: NOT a flick, stays IDLE
  - From CHAPTER_SWIPE or MAP_INTERACT: not eligible for flick promotion

  **POINTER_UP — scrub commit path:**
  - In SCRUBBING with remaining=0: → IDLE with chapterIndex = round(scrubT * (total-1))
  - Test edge: scrubT=0 → chapterIndex=0; scrubT=1 → chapterIndex=total-1; scrubT=0.5 with total=10 → chapterIndex=5 (round(0.5*9)=5)

  **POINTER_UP — MAP_INTERACT recovery:**
  - In MAP_INTERACT with remaining > 0: stays MAP_INTERACT, pointerCount decrements
  - In MAP_INTERACT with remaining=0: stays MAP_INTERACT, pointerCount=0 (idle timer is hook's job, not state machine's)

  **CHAPTER_FLY_DONE:**
  - In CHAPTER_SWIPE → IDLE
  - In any other state: no-op

  **MAP_INTERACT_IDLE:**
  - In MAP_INTERACT with pointerCount=0: → IDLE
  - In MAP_INTERACT with pointerCount>0: no-op (still touching)
  - From other states: no-op

  **TAP_BACKGROUND:**
  - IDLE → PAUSED
  - PAUSED → IDLE
  - From SCRUBBING/CHAPTER_SWIPE/MAP_INTERACT: no-op

  **JUMP_CHAPTER (keyboard / programmatic):**
  - From IDLE with delta=+1: → CHAPTER_SWIPE chapterIndex+1
  - At chapterIndex=total-1 with delta=+1: clamps, no-op (returns same state)
  - At chapterIndex=0 with delta=-1: clamps, no-op
  - From SUSPENDED: no-op (caught by early return)

  **TOGGLE_PAUSED:**
  - IDLE → PAUSED
  - PAUSED → IDLE
  - From other states: no-op

  Use `initialState(10)` as the starting fixture. Create a small helper `function withState(overrides: Partial<ReelState>): ReelState` to reduce boilerplate. Each `it` block: 3-6 lines. Total file should be 200-350 lines.</action>
