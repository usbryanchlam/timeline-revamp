## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Public internet -> VM host port 8787 | Only Nginx (08-02) crosses this. The API container's host port-bind is `127.0.0.1:8787:8787` -- not externally reachable. Public attack surface in 08-01 is zero. |
| Host filesystem -> API container | `.env` arrives via Compose `env_file` directive (process env, not image layer). `.oci/timeline-revamp.pem` arrives via read-only bind mount. Container runs as non-root user `app` (uid 1001). |
| API container -> Postgres container | Internal Docker network only (`postgres:5432` resolves via Compose default network DNS). Postgres has no host port publish in prod. |
| Build context -> image layers | .dockerignore excludes `.env`, `.oci`, `.git`, `.planning` so secrets and untracked dev state never enter image layers. |
| Image registry -> VM | NOT crossed in Phase 8 -- build is local to VM per D-10. No registry exposure until Phase 9. |
| API container -> static `dist/` files | The API serves `/app/dist/` via Hono `serveStatic`. The runtime user (`app`, uid 1001) has read-only access; the dist/ was COPY'd in the runtime stage and inherits root:root ownership (alpine default). Read-only by accident — exactly what we want; the app cannot mutate the bundle. |
